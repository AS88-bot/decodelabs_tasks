# DecodeLabs Cloud Computing Project 1

## Project Overview
This project is part of the DecodeLabs Cloud Computing Internship.

The objective was to host a static portfolio website using AWS S3 Static Website Hosting without using traditional servers.

- GitHub

- Publicly accessible deployment
- Cloud-based infrastructure
